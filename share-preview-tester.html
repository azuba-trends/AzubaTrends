Share Preview Tester

Internal tool — not linked from the storefront, delete or ignore once you're done testing.

This page fetches /api/product?id=YOUR_ID (the Vercel serverless function in api/product.js) and reads back the Open Graph meta tags (og:title, og:description, og:image) it returns — the same tags WhatsApp/Facebook/Twitter read when someone shares a product link.

This only works when this page itself is deployed on Vercel (same place api/product.js runs) — it will NOT work on GitHub Pages or when just opened as a local file, since there's no serverless function to call there. Deploy this file to your Vercel project alongside everything else, then open it at https://your-site.vercel.app/share-preview-tester.html.
