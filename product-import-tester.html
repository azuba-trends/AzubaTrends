Product Import Tester

Internal tool — not linked from the storefront or the admin panel, delete once you're happy with it.

Paste any third-party product page URL below. This calls /api/import-product?url=..., which fetches that page on the server and reads back its og:title, og:description, and og:image tags — the same way a link preview is built. It also downloads the image itself so you can see exactly what would get pulled in.

This is a one-time import helper, not a live sync — it does not keep watching the source page or auto-update price/stock later. Price is deliberately never auto-filled (every site formats it differently); you'll always type that in yourself.

Works on Vercel only — same as api/product.js. Deploy this file alongside your project and open it at https://your-site.vercel.app/product-import-tester.html.
